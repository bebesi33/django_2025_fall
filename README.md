Webprogramming with Django - 2025 Fall

This course covers the basics of Django web-programming.

<h4>Content</h4>
<ul>
  <li>Django project and app structure</li>
  <li>Django template language basics</li>
  <li>Template inheritance and reusable layouts</li>
  <li>Passing data from views to templates</li>
  <li>Static files and URL handling in templates</li>
  <li>Django ORM fundamentals</li>
  <li>Defining models and database relationships</li>
  <li>Database migrations and schema management</li>
  <li>Querying the database with QuerySets</li>
  <li>Django admin interface basics</li>
  <li>Django forms overview</li>
  <li>ModelForms and form handling workflow</li>
  <li>Form validation and error handling</li>
  <li>User authentication system overview</li>
  <li>Login and logout functionality</li>
  <li>User registration process</li>
  <li>Access control and authentication decorators</li>
  <li>Django testing framework basics</li>
  <li>Writing and running tests for models and views</li>
</ul>

<h4>Prerequisites</h4>
Ensure you have the following available:

<li> Python 3.13.3 or later version </li>
<li> Required Python packages (listed in requirements.txt)  </li>
